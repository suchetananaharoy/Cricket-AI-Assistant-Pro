Project Overview:
Cricket AI Assistant Pro is an interactive, web-based application designed to assist cricket enthusiasts, analysts, and team managers in making informed decisions about player selection and match strategy. The application uses HTML, CSS, and JavaScript to provide a modern, responsive, and professional interface.

Key Features

Interactive Chatbot

Ask about team selection, player form, injuries, pitch conditions, weather forecasts, and head-to-head statistics.

Provides real-time responses with actionable insights.

Recommended Playing XI

Generates the best team based on player form, fitness, and historical performance.

Excludes injured players automatically.

Assigns Captain and Vice-Captain badges based on form and performance.

Player Stats & Badges

Displays player role, form, injury status, impact percentage, and badges like In-Form, Captain, or Vice-Captain.

Interactive hoverable impact graph for visual analysis.

Match Strategy Score

Calculates overall team readiness as a percentage based on player form, injuries, pitch, weather, and role balance.

Helps evaluate how strong the team is for the upcoming match.

Role-Based Filtering

Filter recommended players by Batsman, Bowler, or All-Rounder.

Quickly analyze specific player types for tactical decisions.

Export & Copy Functionality

Copy recommended XI to clipboard.

Export team and stats to PDF/print for offline use.

Modern Responsive UI

Dark mode with neon accents for professional appearance.

Rounded cards for dashboards, stats tables, and chat windows.

Fully responsive layout for desktop and tablet devices.

Technologies Used

HTML5 – Structure of the web application.

CSS3 – Styling, responsive design, gradients, hover effects, and modern UI components.

JavaScript – Dynamic functionality, chat responses, team recommendation logic, and interactive graphs.

How It Works

The user can interact with the AI chatbot to request recommendations or information.

The application evaluates player form, role, injuries, pitch conditions, weather, and historical stats.

A recommended XI is generated with badges and impact percentages.

The dashboard displays player stats, impact graphs, and a match strategy score.

Users can filter players, copy the team, or export the data for offline use. 





Potential Future Enhancements

Clone the repository.
